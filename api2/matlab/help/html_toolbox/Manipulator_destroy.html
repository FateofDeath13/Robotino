Manipulator_destroy

Destroys the Manipulator object assigned to ManipulatorId

Contents

  • Function
  • Example Usage
  • See also

Function

[ success ] = Manipulator_destroy( ManipulatorId )

ManipulatorId is the id of the Manipulator object to be destroyed

Example Usage

success = Manipulator_destroy( 0 )

See also
