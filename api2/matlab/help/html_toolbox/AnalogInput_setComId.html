AnalogInput_setComId

Associate an AnalogInput object with a communication interface Bind the AnalogInput to a specific Robotino.

Contents

  • Function
  • Example Usage
  • See also

Function

[ success ] = AnalogInput_setComId( AnalogInputId, ComId )

ComId is the communication id.

AnalogInputId is the id of the AnalogInput object.

Example Usage

success = AnalogInput_setComId( 0, 0 )

See also

AnalogInput Example

function [ success ] = AnalogInput_setComId( AnalogInputId, ComId )
if ( init )
    success = logical( calllib( 'rec_robotino_com_c','AnalogInput_setComId', AnalogInputId, ComId ) );
end;
