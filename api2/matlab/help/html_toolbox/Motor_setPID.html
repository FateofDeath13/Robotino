Motor_setPID

Sets the proportional, integral and differential constant of the PID controller.

Contents

  • Function
  • Example Usage

Function

[ success ] = Motor_setPID( MotorId, Kp, Ki, Kd )

MotorId is the id of the Motor object.

Kp is the Proportional Constant. Typical value 0.95.

Ki is the Integral Constant. Typical value 0.1.

Kd is the differential Constant. Typical value 0.01.

Example Usage

success = Motor_setPID( 0, 0.95, 0.1, 0.01 )

function [ success ] = Motor_setPID( MotorId, Kp, Ki, Kd )
if ( init )
    success = logical( calllib( 'rec_robotino_com_c','Motor_setPID', MotorId, Kp, Ki, Kd ) );
end;
