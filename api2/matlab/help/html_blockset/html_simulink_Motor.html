Motor

Robotino has three motors which it uses for omnidirectional motion. The motors can be controlled individually by providing the Motor Number [0 - 2] . Speed, PID values, brake can all be controlled using the simulink block provided.

Contents

  • Block properties
  • Examples

Block properties

Inputs

  1. ComId
  2. Target speed [rpm]
  3. Brake - set true (1) to activate the brake, set false (0) for otherwise.
  4. Reset Position - to reset the position of the motor.

Outputs

  1. Actual speed - the actual speed (rpm) of a motor.
  2. Actual position - the actual position of a motor.
  3. Motor current - the current (A) of a motor.

Parameters

  1. Motor number - the motor number [0 - 2]
  2. Constant P - the proportional constant. Typical value 0.95.
  3. Constant I - the integral constant. Typical value 0.1.
  4. Constant D - the differential constant. Typical value 0.01.
