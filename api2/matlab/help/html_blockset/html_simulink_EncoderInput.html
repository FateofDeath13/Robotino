EncoderInput

Encoder values such as Position (Actual position of the encoder in ticks since power or Reset) and Velocity (Actual velocity in ticks/s) can be read.

Contents

  • Block properties

Block properties

Inputs

  1. ComId
  2. Reset position - set true (1) to set the current position to zero.

Outputs

  1. Velocity - the actual velocity in ticks/s.
  2. Position - Returns the actual position in ticks since power on or since 'Reset position' was set to true (1).


Published with MATLAB® 7.5
