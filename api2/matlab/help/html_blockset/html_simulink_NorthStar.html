NorthStar

Implements the NorthStar feature on Robotino.

Contents

  • Block properties

Block properties

Inputs

  1. ComId
  2. roomId - Set the id of the room.
  3. ceilingCal - Calibration of the ceiling.

Outputs

  1. sequenceNo - The sequence number.
  2. roomId - The id of the current room.
  3. numSpotsVisible - The number of visible spots.
  4. posX - The current position in x direction.
  5. posY - The current position in y direction.
  6. posTheta - The current orientation in radians.
  7. magSpot0 - The signal strenght of spot 1.
  8. magSpot1 - The signal strenght of spot 2.


Published with MATLAB® 7.5
