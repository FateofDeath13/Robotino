Odometry

Robotino's odometry data can be read and set using the simulink block. Odometry information is provided in the form of X, Y and Phi.

Contents

  • Block properties

Block properties

Inputs

  1. ComId
  2. Set - set true (1) to set the odometry values, set false (0) for otherwise.
  3. x, set - the global x position in mm.
  4. y, set - the global y position in mm.
  5. phi, set - the global phi position in degrees.

Outputs

  1. x - the global x position in mm.
  2. y - the global y position in mm.
  3. phi - the global phi position in degrees.


Published with MATLAB® 7.5
