Manipulator

Robotino can be equipped with a manipulator (arm) and this manipulator can be controlled using the provided simulink block.

Contents

  • Block properties
  • Examples

Block properties

Inputs

  1. ComId
  2. angles - Array containing the set-points in deg.
  3. numAngles - Number of elements in angles.
  4. speeds - Array containing the speed set-points in rpm.
  5. numSpeeds - Number of elements in speeds.

Outputs

  1. seq - The sequence number.
  2. angles - Array storing the current axes positions in degrees.
  3. numAngles - The number of elemets stored in angles. This is equal to the number of axes.
  4. speeds - Array storing the current axes speeds in rpm.
  5. numSpeeds - The number of elemets stored in speeds. This is equal to the number of axes.
  6. errors - Error code of axes.
  7. numErrors, The number of elemets stored in errors. This is equal to the number of axes.
  8. motors_enabled, Is 1 if the motors are enabled. Otherwise 0.
  9. store_current_position, Is 1 if the store position button is pressed. Otherwise 0.
  10. cwAxesLimits, Array storing the current axes limits in degrees.
  11. numCwAxesLimits, The number of elemets stored in cwAxesLimits. If axes limits had been received this is equal to the number of axes. Otherwise 0.
  12. ccwAxesLimits, Array storing the current axes limits in degrees.
  13. numCcwAxesLimits, The number of elemets stored in ccwAxesLimits. If axes limits had been received this is equal to the number of axes. Otherwise 0.
